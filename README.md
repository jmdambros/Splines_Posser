Trata-se de um projeto que utiliza os splines para fazer o caminho dos inimigos em um jogo de Tower Defense. No caso, replicamos a mecânica de trajeto dos balões no jogo Bloons TD.
Quanto às mecânicas de funcionamento, clicar barra da espaço faz balões nascerem, que irão seguir um spline de trajeto. Os balões podem ser estourados ao clicar neles com o mouse.
